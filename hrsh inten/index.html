Weather Dashboard

I developed a Weather Dashboard to provide users with real-time weather updates and forecasts for any location. The dashboard features a clean and intuitive interface that adapts to different screen sizes for optimal usability on both desktop and mobile devices. Utilizing APIs for fetching weather data, this project showcases my ability to integrate third-party services and work with asynchronous JavaScript. The Weather Dashboard demonstrates my proficiency in HTML, CSS, and JavaScript, and my commitment to building practical, user-friendly web applications.

Project Title 2

To-Do List App

I created a To-Do List App to help users manage their daily tasks and improve productivity. The app allows users to add, edit, and delete tasks, and mark them as complete. Key features include a user-friendly interface for easy task management, task categorization to organize tasks by priority or type, and due date and reminder functionality to ensure timely task completion. The app uses local storage to save tasks between sessions without requiring an account, providing a seamless user experience. Developed using HTML, CSS, and JavaScript, this project demonstrates my skills in front-end web development and my ability to create functional, user-centric applications.
